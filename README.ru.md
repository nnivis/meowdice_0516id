
# Dice Battle

🇬🇧 [Read in English](README.md)

Пошаговая стратегия с кубиками для двух игроков на Unity. Бросай кубики, заполняй поле 3×3 и набирай больше очков, чем соперник.

---

## О проекте

Dice Battle — портфолио-проект с акцентом на чистую слоистую архитектуру в Unity. Игроки ходят по очереди — каждый ход активный игрок бросает кубик (1–6) и ставит его в свободную клетку своего поля 3×3. Когда оба поля заполнены, считаются очки по колонкам и определяется победитель. Совпадения значений в колонках соперника запускают специальные механики сброса, добавляя тактический элемент поверх случайности.

Режим ВС ИИ полностью реализован. Локальный PvP и онлайн PvP — заготовки для дальнейшей разработки.

---

## Механики

- Бросить кубик и поставить его в любую свободную клетку поля 3×3
- Заполнить колонку одинаковыми значениями — и та же колонка соперника очищается
- Формула очков колонки: `значение × количество²` — комбо усиливаются экспоненциально
- Соперник-ИИ со случайной стратегией размещения
- Кастомизация скина кота для своего персонажа
- Экран результата матча после каждой игры

## Архитектура

| Паттерн | Реализация |
|---|---|
| Dependency Injection | VContainer — `RootLifetimeScope` / `GameplayLifetimeScope` |
| State Machine | `MainSceneMode` — состояния Main, Gameplay, Result |
| Слоистая архитектура | Domain → Services → Infrastructure, без обратных зависимостей |
| Асинхронный игровой цикл | `GameFlow` / `TurnSystem` — `async/await` + `CancellationToken` |
| Слой данных | `IPersistentData` / `IDataProvider` — JSON через Newtonsoft.Json |

## Запуск

1. Клонировать репозиторий
2. Открыть в Unity 6000.3 или новее
3. Открыть `Assets/Scenes/BootstrapScene`
4. Установить VContainer через Package Manager: `jp.hadashikick.vcontainer`
5. Нажать Play
